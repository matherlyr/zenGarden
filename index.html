WHAT?? ZEN GARDEN!! Cool..

  • Zen Garden is an interesting website beacuse you can view the same website in many different visual formats.
    This is done by editing the CSS of the webpage so that the information can be presented in a way that makes sense
    Zen Garden encourages users to create their own version of the website in order to display all the wonderful things
    that you can do with CSS.
  • One of the CSS formats I really like is called "A Robot Named Jimmy." Jimmy is always smiling, and it makes for a happy visit
    to the site each time I visit. One of the features that I like on Jimmy's website is the fact that it has a liquid layout,
    meaning that no matter the size of the screen, you can always see Jimmy's happy face and the content he wants to share.
